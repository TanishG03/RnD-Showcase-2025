How to Read the HEIDI Matrix Visualization

  1. Understand the Structure: The HEIDI matrix is a grid where each cell represents the relationship between two data points. The matrix is divided into blocks based on cluster labels.
  2. Cluster Blocks: Each block corresponds to interactions between points from specific clusters. For example, the top-left block shows interactions within Cluster 0.
  3. Colors and Expressions: Each color represents a unique boolean expression (e.g., F0 + F1). The legend maps colors to expressions.
  4. Spiral Placement: Expressions are arranged in a spiral pattern, with the most frequent and significant patterns at the center.
  5. Scatter Plot (Optional): If included, the scatter plot shows the dataset in 2D space, with points colored by cluster assignments.
